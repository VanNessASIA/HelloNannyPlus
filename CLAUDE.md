# HelloNanny プロジェクト指示書

## デザインシステム
UI/UXを実装する際は、必ず以下のデザインガイドラインを参照してください:
→ **HELLONANNY_DESIGN_SYSTEM.md**

このガイドラインには、カラー、タイポグラフィ、コンポーネント、アニメーション、
トーン＆マナーなど、HelloNannyブランドのすべてのデザインルールが定義されています。

## プロジェクト構成
- `/` — HelloNanny+ LP（Webアプリ用ランディングページ）
- `/hellonanny-main/` — HelloNanny本体サイト（マルチページ）

## 技術スタック
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- lucide-react（アイコン）
- 4言語対応（EN/JA/ZH/TH）

## デプロイ
- Vercel を使用
- HelloNanny+: https://hello-nanny-plus.vercel.app
- HelloNanny Main: https://hellonanny-main.vercel.app
